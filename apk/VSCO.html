Fitur Mod Aplikasi

Mengaktifkan Anggota Pro

Member atau anggota pro telah saya aktifkan, tanda jika pro sudah aktif yaitu akan ada badge atau logo pro di bagian poto profil mu.

Membuka Semua Filter/Efek

Filter dan Efek sudah terbuka semuanya secara gratis, jadi mengeditlah dengan nyaman.

Mengaktifkan Fitur Beta

Fitur Beta sudah ku munculkan, jadi akan ada beberapa tambahan seperti bagian Tools/Alat, Efek, dan Filter baru.

Fix Login Issue

Memperbaiki masalah login (hanya berfungsi pada bagian Facebook dan Email, selain dua ini kemungkinan akan mengalami error login).
